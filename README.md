# Skills For Real Engineers

[![skills.sh](https://skills.sh/b/ishenli/skills)](https://skills.sh/ishenli/skills)

这是我为真实工程工作定制的一套 agent skills 仓库。它的目标是让 AI 协同日常工作更可控、更稳定，而不是依赖“vibe”式的试错。

## 快速开始（30 秒设置）

1. 运行 skills.sh 安装器：

```bash
npx skills@latest add ishenli/skills
```

## 为什么这些技能要存在

这是一个面向真实工程的 Agent skills 仓库。它的目标是让 AI 协同开发更可控、更稳定，而不是把“流程”变成一个黑盒。


- **boss-review** — 对工作文档、方案、周报、复盘等进行严格审计。
